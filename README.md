# Counter App 🔢

A simple and interactive counter application built using HTML, CSS, and JavaScript.
Users can increase, decrease, and reset the counter value with button clicks.

---

## 🚀 Features

* Increase the counter value
* Decrease the counter value
* Reset the counter to zero
* Simple and clean user interface
* Real-time updates

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation)

---

## 📂 Project Structure

* `index.html` → Structure of the app
* `counter.css` → Styling of the UI
* `counter.js` → Functionality (Increase, Decrease, Reset)

---

## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Double-click on `index.html`
4. Use the buttons to interact with the counter

## 🎯 How It Works

* Click **Increase** → number increases by 1
* Click **Decrease** → number decreases by 1
* Click **Reset** → number becomes 0
